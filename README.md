# NullSteve

Хоррор-мод для Minecraft 1.21.1 (Fabric, Java 21).

NullSteve — это сущность, которая постепенно появляется в твоём мире.
Чем дольше ты играешь, тем чаще она будет рядом.

## Стадии хоррора

1. **Стадия 1 — Присутствие.** Странные системные сообщения, мерцание частиц.
2. **Стадия 2 — Наблюдение.** Появления за спиной, ложные UI-ошибки.
3. **Стадия 3 — Взаимодействие.** Близкие появления, шёпот в субтитрах.
4. **Стадия 4 — Противостояние.** Прямой контакт, предупреждения "Не оставайтесь".

Стадии переключаются автоматически по времени игры.

## Безопасность

Мод **не делает** ничего разрушительного:
- никогда не удаляет предметы
- никогда не повреждает сохранение
- никогда не блокирует ввод навсегда
- никогда не вызывает падения клиента
- сущность неуязвима, но исчезает сама через 4000 тиков, если её никто не видит
- сущность исчезает в момент, когда игрок прямо на неё посмотрит (поведение настраиваемо)

## Зависимости

- Minecraft 1.21.1
- Fabric Loader
- Fabric API 0.102.1
- YACL 3.5.0
- Mod Menu 11.0.3
- Java 21

## Сборка

```bash
./gradlew build
```

JAR появится в `build/libs/`.

## Настройка

Открой Mod Menu → NullSteve, чтобы настроить интенсивность, интервалы и отдельные эффекты.
Файл конфигурации хранится в `config/null_steve.json`.

## Язык

Только русский (`ru_ru`).
